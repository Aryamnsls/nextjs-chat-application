# Next.js AI Chat Application

An AI-powered chat application built with **Next.js** and **OpenAI API**.  
This project demonstrates modern web development practices and real-time AI interaction.

## Features

- Real-time AI chat responses
- Streaming responses using OpenAI API
- Modern responsive UI
- Optimized API handling
- Fast Next.js architecture

## Tech Stack

- Next.js
- React
- OpenAI API
- Tailwind CSS
- Node.js

```bash
git clone https://github.com/yourusername/nextjs-chat-application.git
